RESPONSIVE WEB DESIGN

icarebnb

icarebnb is a project which three other junior web developers and myself produced using Ruby on Rails in the final 2 weeks of the Le Wagon London coding bootcamp. The aim was to create a web app to connect charity workers to individuals who are willing to host a spare room to help those in need.

I was responsible for front-end development of the website. This involved designing the logo and assets in Illustrator, choosing colour scheme and the site layout.

Bailey Coaching

This is a website I designed and created using React for a personal trainer in Canterbury.

The client wanted to create an online platform to advertise, but with no visual identity other than a logo I designed, I was tasked to create an initial concept using the content provided.

My Biz

I have worked with Canterbury City Council on a few web projects over the years with varying degrees of involvement in development and design, some examples include Buy in the Bay, StartMyBiz and GrowMyBiz.

For MyBiz I was responsible for adding content to the website, the layout of some of the pages and designing some of the imagery found throughout.

The aim of the project was to provide a directory of resources containing useful information to help local businesses.

Analyser Accounting

This is a website I designed and created for a local accounting company.

The client needed a place to display the services they offer and also to add links and documentation for users to download instructions for their software.
